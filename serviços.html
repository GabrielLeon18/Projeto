Serviços e produtos

Nossos serviços:

  • Pacote upgrade

    • Nesse pacote, realizamos upgrade no software que a empresa já possui.

    Garantia de 90 dias.

  • Pacote software

    • Neste pacote, criamos um software para a empresa, totalmente personalizado conforme solicitação.

    Garantia de 90 dias + manutenção de emergência.

  • Pacote completo

    • Neste pacote, também criamos um software para a empresa, no entanto, fazendo uma análise prévia das necessidades da empresa, para então, embutir no projeto final.

    Garantia de 12 meses + manutenções mensais ou de emergência.

    Garantia de 90 dias.

  • Limpeza do sistema

    • Contratando esse serviço, realizamos a limpeza do sistema, tornando-o mais seguro e rápido.

    Garantia de 90 dias.

  • Mapeamento de software

    • Contratando esse serviço, realizamos o mapeamento completo de seu software, e localizamos possiveis riscos e oportunidades para melhorias.

    Garantia de 90 dias.
