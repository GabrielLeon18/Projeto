Serviços e produtos

Nossos serviços:

  • Pacote upgrade

    • Nesse pacote, realizamos um upgrade no software que a empresa já tem.

    Garantia 3 meses.

  • Pacote software

    • Nesse pacote, criamos um software para a empresa, atendendo seu pedido.

    Garantia 3 meses e manutenção de emergencia.

  • Pacote completo

    • Nesse pacote, criamos um software para a empresa, atendendo seu pedido e após analise de sua necessidade.

    Garantia 1 ano e manutenção mensal ou de emergencia.

Nossos serviços adicionais:

  • Correção de software

    • Contratando esse serviço, realizamos a correção de erros em seu software.

    Garantia 3 meses.

  • Relatório do sistema

    • Contratando esse serviço, realizamos uma analise no seu software e mandamos um relatório dele.

    Garantia 2 semanas.

  • Backup de dados do sistema

    • Contratando esse serviço, realizamos um backup dos dados de seu sistema, para que não perca seus dados.

    Garantia 3 meses.

  • Limpeza do sistema

    • Contratando esse serviço, realizamos a limpeza do sistema, tornando seu equipamento mais seguro e rápido.

    Garantia 3 meses.

  • Mapeamento de software

    • Contratando esse serviço, realizamos o mapeamento completo de seu software, onde localizamos possiveis riscos e oportunidades para melhorias.

    Garantia 3 meses.
